# 健身减脂运动管家微信小程序
一款基于 uni-app + SpringBoot 开发的微信小程序，聚焦健身减脂场景，为用户提供「运动+饮食」一体化的智能管理服务，解决健身缺乏规划、饮食把控难的核心痛点。

## 核心功能
### 1. 个性化账号体系
- 微信一键免注册登录，降低使用门槛；
- 支持填写身高、体重、运动目标（减脂/增肌/塑形）等个性化信息，为后续服务提供数据支撑。

### 2. 定制化运动管理
- 根据用户身体数据和运动需求，自动生成一周定制化运动计划（含中等强度、高强度、力量训练）；
- 提供慢跑、跳绳、骑行等常见运动教学视频，支持运动计时、完成情况统计及能量消耗计算；
- 可自定义运动提醒时间，微信实时推送提醒，督促用户完成运动计划。

### 3. 饮食能量管控
- 覆盖碳水、蛋白、脂肪、果蔬四大类常见食物，可查询食物营养成分及能量值；
- 智能推荐早中晚三餐饮食方案，自动计算每日能量摄入及营养占比；
- 以图表+数据形式直观展示饮食结构，帮助用户合理控制减脂期能量摄入。

### 4. 数据统计与互动激励
- 实时统计运动时长、能量消耗、达标天数等数据，生成周/日运动分析报告；
- 运动指数累计机制，支持查看好友/全网排行榜，增强健身趣味性；
- 好友管理功能，可添加/删除好友、查看好友运动数据，实现互动监督。

## 快速运行说明
1. 前端部署：使用HBuilder X打开uni-app项目，配置微信小程序开发者工具路径，运行至模拟器或真机；
2. 后端部署：使用IntelliJ IDEA打开SpringBoot项目，配置MySQL数据库连接信息，启动后端服务；
3. 前后端联调：确保前端请求地址与后端服务地址一致，接口通信正常即可使用。

## 项目价值
小程序兼顾实用性与易用性，无需专业健身知识即可完成科学的运动和饮食规划，帮助用户高效达成健身减脂目标，适配微信生态，可直接面向C端用户使用。
