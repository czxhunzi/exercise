# 健身减脂运动管家微信小程序
基于 uni-app + SpringBoot 开发的健身减脂管家微信小程序，为用户提供一站式健身减脂服务。

## 核心功能
1. 便捷登录与信息管理：支持微信一键免注册登录，可填写身高、体重、运动目标等个性化信息，为后续服务提供数据支撑；
2. 个性化运动计划：根据用户身体信息和运动需求，生成定制化周运动计划，支持计划修改、运动提醒及完成情况统计；
3. 运动教学与数据追踪：提供慢跑、跳绳等常见运动教学视频，支持运动计时，实时统计运动时长、能量消耗等数据；
4. 饮食能量管理：涵盖碳水、蛋白等多类食物营养查询，智能推荐三餐饮食，自动计算每日能量摄入及营养占比；
5. 互动激励功能：累计运动指数生成排行榜，支持添加好友、查看好友运动数据，多维度激励坚持健身。

## 快速运行
1. 前端：HBuilder X 打开项目，运行至微信小程序模拟器；
2. 后端：配置MySQL连接后，启动SpringBoot服务；
3. 联调：确保前后端接口地址一致即可使用。
